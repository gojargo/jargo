# jargo

A Go port of [Pipecat](https://github.com/pipecat-ai/pipecat) — a framework for
building real-time voice and multimodal AI pipelines.

> **Status:** early work in progress. APIs are unstable and will change.

## Install

```sh
go get github.com/gojargo/jargo
```

## About the name

`jargo` (from *jargon* → language/speech) is an independent Go implementation
inspired by Pipecat's pipeline architecture. It is **not** affiliated with,
sponsored by, or endorsed by Daily. "Pipecat" is referenced only to credit the
upstream project this work is derived from.
